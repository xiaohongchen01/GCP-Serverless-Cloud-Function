# Azure
## Steps
* go to Function App
* Click create and select Consumption
* Select python for runtime
* Leave region to whereever defaulted
* Review and create
* Go to resources  and create a function 
* Select http trigger and create
* Edit your code and save
* Function URL will be your URL
